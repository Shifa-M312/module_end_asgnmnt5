MERN Shop Project

A full-stack e-commerce website built for the Module 5 Assignment.

Key Features
- User Login & Register: Secure authentication using JWT.
- Product Catalog: A clean grid of products with search and filtering.
- Shopping Cart: Add/Remove items using Redux Toolkit.
- AI Suggestions: Product recommendations based on RapidMiner logic.
- Responsive Design: Professional layout using Tailwind CSS.

 Tech Stack
- Frontend: React, Redux, Tailwind.
- Backend: Node.js, Express, MongoDB.

 How to Run
1. Backend: 
   - Go to `backend` folder and run `npm install`.
   - Start with `npm run dev`.
2. Frontend: 
   - Go to `frontend` folder and run `npm install`.
   - Start with `npm run dev`.
